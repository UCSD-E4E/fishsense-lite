# FishSense Data Processing Workflow Worker
This repo contains the code for the data processing workflow. It is expected to be run on Waiter, the Junkyard Cluster, the Qualcomm Cluster, or Nautilus.

## How to run
1. Make sure you have `uv` installed. https://docs.astral.sh/uv/getting-started/installation/
1. Run `uv sync`
